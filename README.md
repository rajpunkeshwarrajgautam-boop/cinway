# 🚀 Antigravity Streaming (Netflix Clone)

This is a premium, full-stack Next.js application built autonomously via God Mode. It implements a stunning, zero-placeholder streaming interface utilizing deep aesthetics, responsive design, and CSS custom properties per the strict Antigravity global coding rules.

## ✨ Features
- **God-Tier UI/UX**: True glassmorphism, responsive behavior, custom scrollbars, and `cubic-bezier` hover states.
- **Zero Placeholders**: Uses a rich static proxy dataset simulating an active database connection.
- **App Router**: Utilizes Next.js 15 App router with Server Components.
- **Prisma & NextAuth Ready**: The configuration and database schema for SQLite / PostgreSQL is already scaffolded.

## 🛠️ Tech Stack
- **Framework:** Next.js 15 (TypeScript)
- **Styling:** Vanilla CSS Custom Properties (Strict No-Tailwind Policy)
- **Icons:** Lucide React
- **Data ORM:** Prisma
- **Auth:** NextAuth (Prepared)
